Welcome to MY WEB TECH PROJECT

This web application is a modern multi-user dashboard platform designed to manage users, roles, and permissions. It features dynamic graphs, an interactive calendar, real-time map integration, and a secure login system for admins and users. Built with Node.js, Express, and Bootstrap, it combines clean design with practical functionality — making it an ideal project for learning full‑stack web development.
